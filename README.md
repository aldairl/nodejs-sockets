# SERVER SOCKETS NODEJS

A basic NodeJS server for sockets connections using socket.io library

## Setup

## copy the .env.exmaple file

Replace PORT variable

´´´bash
cp .env.example .env
´´´

## Install dependencies

´´´bash
yarn install
´´´

## run App

Development mode

´´´bash
yarn dev
´´´
Production mode

´´´bash
yarn start
´´´

## display frontend

Go to http:// localhost:PORT

Enjoy!
